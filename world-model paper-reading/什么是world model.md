# 4D生成

简单来说，4D生成的结果是3D+时间。3D生成的结果更像雕塑，我们可以从各个角度观察这个物体，但是物体本身不具备时间信息；而4D生成的结果在3D的基础上还多了：速度，角速度，以及该点在时间轴上的存在时间。3D和4D生成的对比如下：

| **特性**   | **普通 3D 资产 (Static 3D Assets)**                            | **NeoVerse 的 4D 资产 (Dynamic 4D Gaussians)**                                             |
| -------- | ---------------------------------------------------------- | --------------------------------------------------------------------------------------- |
| **本质**   | **空间快照**。它只捕捉某个瞬间的几何和纹理。                                   | **时空连续体**。它同时捕捉了空间结构和时间上的动态变化。                                                          |
| **运动能力** | **本身是静止的**。如果要动，需要人工绑定骨骼（Rigging）或制作关键帧动画。                 | **自带运动信息**。每个点都携带速度和角速度参数，模型本身就包含“动势” 。                                                 |
| **时间插值** | 无法直接插值。如果只有第1帧和第10帧的 3D 模型，很难通过简单的线性插值得到第5帧（可能会出现模型穿插或破裂）。 | **支持平滑插值**。利用内置的速度参数，可以计算出任意中间时刻（比如 $t=1.5$）的状态，从而轻松实现**慢动作（Slow-motion）**或**子弹时间**效果 。 |
| **数据形态** | 通常是多边形网格（Mesh）或静态点云。                                       | 动态的高斯球集合（Gaussian Splatting），其属性随时间 $t$ 变化 。                                            |
